# Cure Pathology, Muzaffarpur - Website

A modern, responsive, and user-friendly website designed and developed for **Cure Pathology**, located in Muzaffarpur, Bihar. This platform aims to digitalize the lab's presence, allowing patients to explore available tests, health packages, and get in touch seamlessly.

---

## 🚀 Features

*   **Fully Responsive Design:** Optimized for seamless viewing across all devices (Mobiles, Tablets, and Desktops).
*   **Test Catalogue:** Clean display of available pathology tests, diagnostic services, and health checkup packages.
*   **Easy Booking/Inquiry:** Integrated call-to-action buttons for quick appointments and inquiries via WhatsApp or phone.
*   **Location Integration:** Embedded Google Maps navigation to help patients find the lab effortlessly in Muzaffarpur.
*   **Performance Focused:** Lightweight, clean code ensuring fast loading times.
